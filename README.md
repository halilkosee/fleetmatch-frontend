# FleetMatch Frontend

React + Vite frontend for the EasyFleetMatch platform.

## Stack

- React
- TypeScript
- Vite
- React Router
- Axios
- pnpm

## Local development

```bash
pnpm install
pnpm dev
```

Create a local `.env` file from `.env.example` and set `VITE_API_BASE_URL` to the backend API URL.
